오이 계란 샌드위치

오이와 계란을 활용한 간단한 샌드위치

인분1인분
조리 시간15분
난이도easy

필요한 재료

  • 식빵 2개
  • 오이 1개
  • 계란 2알
  • 마요네즈 2큰술

현재 있는 재료

  • 식빵: 필요 2개 / 보유 10개
  • 오이: 필요 1개 / 보유 보유량 모름
  • 계란: 필요 2알 / 보유 2알

없는 재료

  • 마요네즈 2큰술

선택 재료

없음

조리 도구

  • 칼
  • 도마
  • 볼
  • 숟가락

조리 기기

  • 가스레인지

조리 순서

  1. 계란을 삶아 껍질을 벗긴 뒤 볼에 넣고 으깬다.
  2. 오이를 얇게 썬다.
  3. 으깬 계란에 마요네즈를 넣고 섞는다.
  4. 식빵 위에 오이와 계란 속을 올린다.
  5. 다른 식빵으로 덮어 샌드위치를 완성한다.
