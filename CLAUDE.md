# food_agent

## 프로젝트 개요

Groq Llama 4 Scout 기반 음식 사진 레시피 생성 Streamlit 앱.
- 앱 진입점: `src/main.py`
- 실행: `streamlit run src/main.py`
- 환경변수: `.env`에 `GROQ_API_KEY` 필요

## 하네스: food-agent 개발

**목표:** 음식 사진 분석·레시피 생성 앱의 기능 개발·개선을 에이전트 팀으로 가속화한다.

**트리거:** food-agent 기능 추가, 수정, 개선, 디버깅, 코드 리뷰 요청 시 `food-agent-dev` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
