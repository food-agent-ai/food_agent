# Security Policy

## Supported Versions

This project is under active development. Security fixes, when applicable, are generally applied to the latest version of the default branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please do not disclose it publicly right away.

Instead, please report it privately to the repository maintainers with:
- A description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix, if available

We will review the report as soon as possible and take appropriate action.
